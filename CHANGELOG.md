# Changelog

All notable changes to Oliver are documented here.

---

## [1.0.0] — March 2026

### 🎉 Initial Release

**Core Features**
- Book catalogue with fuzzy search powered by Fuse.js
- AI research chat powered by Claude AI
- Mood-based book recommendations
- Reading challenge tracker with progress milestones
- Community book reviews with star ratings
- Staff admin panel for catalogue and challenge management
- Dark mode with system preference support
- Google Authentication for `@abingdon.org.uk` accounts
- Email/password login as fallback

**Technical**
- Single-file SPA — all CSS and JS inline in `index.html`
- Firebase Auth, Cloud Firestore, and Firebase Analytics integration
- All dependencies self-hosted on GitHub Pages to bypass Smoothwall content filtering
- Deployed on GitHub Pages at `whatsgoodiemygang.github.io/oliver`

---

## How Versions Work

This project uses [Semantic Versioning](https://semver.org/):
- **Major** (1.x.x) — big new features or breaking changes
- **Minor** (x.1.x) — new features that don't break anything
- **Patch** (x.x.1) — bug fixes and small tweaks
