# Contributing to Oliver

Thanks for your interest in contributing to the Abingdon School Library app. Here's how to get involved.

---

## Who Can Contribute

This project is maintained by Seb and is built specifically for Abingdon School. Contributions are welcome from:
- Abingdon School staff and pupils
- Anyone with permission from the project maintainer

---

## Ways to Contribute

### 🐛 Reporting a Bug
1. Go to the [Issues](https://github.com/whatsgoodiemygang/Oliver/issues) tab
2. Click **New Issue** and select the Bug Report template
3. Fill in as much detail as possible — what happened, what you expected, and any console errors

### 💡 Suggesting a Feature
1. Go to the [Issues](https://github.com/whatsgoodiemygang/Oliver/issues) tab
2. Click **New Issue** and select the Feature Request template
3. Describe what you'd like and why it would be useful

### 🛠️ Making Changes
1. Fork the repo
2. Make your changes to `index.html`
3. Test locally by opening the file in a browser (note: Firebase Auth won't work locally — use a local server like VS Code Live Server)
4. Open a Pull Request with a clear description of what you changed and why

---

## Code Style

Since the entire app lives in one HTML file, keep it consistent:
- CSS goes inside the `<style>` block at the top
- JavaScript goes inside the `<script>` block at the bottom
- Use clear comments to label new sections
- Keep the pink brand colour (`#DE048E`) consistent throughout

---

## Questions

If you're unsure about anything, open a Discussion or reach out via the repo.
